18. Перевантажити операції <= (порівняння часу за абсолютним значенням різниці у хвилинах від поточного часу), += (збільшення на n хвилин), відсортувати масив екземплярів класу часів за зростанням абсолютного значення різниці від поточного часу з використанням алгоритму сортування обміном, збільшити кожен час в масиві на задану кількість хвилин, перевірити, чи часи у масиві далі будуть впорядковані.
